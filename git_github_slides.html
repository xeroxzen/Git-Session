Essential Git Commands üíª

# First-time setup git config --global user.name "Your Name"
git config --global user.email "you@example.com" # Starting a project
git init # Initialize repository
git status # Check current state

git pull

Connecting Local to Remote

# Connect your local repo to GitHub git remote add origin https://github.com/username/repo-name.git # Push your code to GitHub git push -u origin main # Future pushes (after first time) git push # Get updates from GitHub git pull

Pro Tips:

  • Always pull before you push
  • Use SSH keys for easier authentication
  • Keep your main branch clean
  • Use descriptive commit messages
